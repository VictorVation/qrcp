# qrcp

Quickly transfer files over your wifi network using a QR code or URL, straight from your terminal!

## Usage

Install it from NPM (requires Node >= 8):

```bash
npm install -g qrcp
```

Run it from your terminal:

```bash
qrcp <filename>
```
